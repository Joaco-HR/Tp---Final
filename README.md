# Tp---Final
Este programa se trata de la base de datos de una comiquería que permite ver el catálogo, gestionar los productos, realizar pedidos, ver el historial de los últimos 5 productos vistos y        visualizar los productos por categorías.

Cómo usarlo:
- Para ejecutarlo, cargá el archivo en Python o en Visual Studio Code y presioná F5.
- Una vez en funcionamiento, podés navegar por el programa escribiendo con el teclado la opción que querés elegir y lo que querés ver o realizar.

Decisiones de diseño:
- Decidimos usar JSON para el almacenamiento de los diferentes datos, con el fin de tener un código mejor organizado y más profesional.
- Utilizamos la estructura HashTable para el apartado de gestión de productos, ya que su estructura resultó cómoda para trabajar.
- Para los apartados de historial de visualización y categorías, optamos por un enfoque más personal, desarrollándolos de la manera que nos resultó más intuitiva.
- En el apartado de pedidos, decidimos usar Queues (colas), debido a que sus funciones facilitaron el desarrollo de esta parte.
